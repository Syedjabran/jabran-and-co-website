Allocation Methods

How each monthly cost pool is distributed across orders. Overhead pool = recurring fixed expenses + subscriptions (PKR) + the month's overhead ledger entries. Payroll pool = active payroll (PKR) minus direct labour already charged through timesheets.

Manual Weights

Used only when a pool's method is Manual Weights. Values are normalised — 30/50/20 and 3/5/2 give identical results.

Run Monthly Allocation

Distributes both pools across orders using the methods above. Re-running a month replaces that month's results — nothing is duplicated. PKR only: non-PKR lines, employees, or orders are excluded and reported below, never silently converted.
